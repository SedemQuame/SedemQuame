Opportunity

Renmo wanted a trustworthy digital front door for a complex fintech product. Customers needed education about rent-to-own mechanics, eligibility, and documentation requirements—all while keeping payment flows and landlord onboarding smooth across devices with limited bandwidth.

Experience design

  • Built modular UI components using an in-house design system inspired by Material guidelines.
  • Implemented accessibility checks (aria labels, focus states, color contrast) for AA compliance.
  • Localized content and form logic to support regional regulation differences.

Platform integration

  • Connected the React front end to Node.js + PostgreSQL services for underwriting and payment schedules.
  • Added analytics + CRM hooks so growth teams tracked funnels without relying on engineering.
  • Containerized the app for AWS + Heroku deployments with automated rollbacks.
